# 📚 Library Management System Using Java Legacy Classes

## 📖 Project Description

This is a simple console-based Library Management System developed using Java.  
The project demonstrates the use of Java Legacy Classes such as:

- Vector
- Hashtable
- Stack
- Enumeration
- Date
- Calendar

The system allows users to:

- Add books
- Display books
- Search books
- Issue books
- Return books

---

# 🚀 Features

✅ Add New Books  
✅ Display All Books  
✅ Search Book by ID  
✅ Issue Book  
✅ Return Book  
✅ Menu Driven Program  
✅ Uses Java Legacy Classes
